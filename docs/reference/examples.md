# Examples

This page groups concrete naming examples by infrastructure area.

```text

zrh-lan-prd-core-dc-02-g1
```

## Data Platform

```text
zrh-lan-prd-data-db-01-g1
zrh-lan-prd-data-db-02-g1
```

## Web Platform

```text

```

## Monitoring


## Multi-location Server Farm Examples

These examples show how the same naming standard can be applied across multiple colocation or hosting locations using city-based location codes only.

### Frankfurt

```text
fra-lan-prd-core-vmh-01-g1
fra-lan-prd-data-db-01-g1
fra-dmz-prd-edge-web-01-g1
```

### Gravelines

```text
gvl-lan-prd-core-vmh-01-g1
gvl-lan-prd-data-stor-01-g1
gvl-dmz-prd-edge-lb-01-g1
```
